## FlaskAPI-Skeleton
Flask API Server Written in Python


#### Virtual Environments

* skeletonenv


#### Configurations

* config.json (required)
	* host
	* port

* dev_config.json (optional)
	* debug


#### Running The Code

	$ source skeletonenv/bin/activate
	$ python3 flask_skeleton.py

#### Stopping the Code

	$ ^C (Ctrl + C)
	$ deactivate (to get out of virtual environment)
